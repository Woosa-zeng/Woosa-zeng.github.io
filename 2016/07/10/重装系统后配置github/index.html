重装系统后如何配置git本地与远程仓库

前几天系统重装了，很多东西都没有备份，工作快三个月了，由于公司用svn,不用git，对git的东西有点生疏了，网上找了些贴子，自己整理一下，方便以后如果再次重装

配置ssh

ssh-keygen -t rsa -C “your email@example.com”

邮箱是你GitHub上留的邮箱，需要输入时直接回车，rsa就生成好了。
找到你在C盘的.ssh文件夹，打开id_rsa.pub,复制密码。
打开远程GitHub Settings –>SSH and GPG keys –>New SSH Key之前的没用了可以直接删掉。
然后就可以在本地pull了

配置hexo

安装git
安装nodejs
然后重新下载hexo npm install -g hexo
如果之前的hexo文件夹还在，就可以直接hexo new name写新的文章了
如果什么都没有了，可以看看这个
